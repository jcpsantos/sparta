.. py:function:: get_secret_aws(name: str, region: str = 'sa-east-1') -> Dict[str, Any]

   Function to retrieve data stored in AWS Secrets Manager.

   :param name: The name of the secret to retrieve.
   :type name: str
   :param region: The name of the region where the secret is stored. Defaults to 'sa-east-1'.
   :type region: str, optional

   :returns: A dictionary with the retrieved data.
   :rtype: dict

   .. rubric:: Example

   >>> secret_data = get_secret_aws('my_secret', 'us-west-2')
