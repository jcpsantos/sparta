.. py:function:: get_secret_aws(name: str, region: str = 'sa-east-1') -> dict[Any, Any]

   Function to capture the data registered in Secret Manager.

   :param name: Secret Manager name.
   :type name: str
   :param region: Region code where Secret Manager is located. Defaults to "sa-east-1".
   :type region: str, optional

   :returns: Dictionary with the captured data
   :rtype: dict

   .. rubric:: Example

   >>> get_secret_aws('Nome_Secret', 'sa-east-1')
